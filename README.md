# rename-subtitle

## 설명

동영상 파일, 자막 파일의 이름을 동일하게 변경해 주는 프로그램 입니다.

개인 공부 겸 지인 배포용도로 제작했습니다.

사용하실 분들은 release에서 최신 버전으로 다운로드 하시면 되고 

Windows 타겟으로만 개발했기 때문에 소스코드 변경없이 다른 타겟으로 빌드했을 때

정상적인 동작이 불가능할 수 있습니다.
